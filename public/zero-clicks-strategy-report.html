The Problem

Pubs are invisible on Google

83% of independent pubs haven't posted on their Google Business Profile in 90+ days. Unanswered reviews signal to tourists that nobody is home. Competitors who post weekly appear 2.8× more in local search.

The Solution

Automated presence management

Zero Clicks runs two AI-powered engines: one that replies to every Google review in the pub's voice, and one that posts weekly offers and updates to attract walk-in traffic — all with zero admin from the owner.

Core Principle

"AI writes the message. The pub controls the economics." — All offers are pre-approved by the pub owner. AI only writes the presentation, never invents the discount.

Core Product

Reputation Manager — How It Works

A fully automated Google review reply system. Once set up, the pub owner never touches it. Claude replies to every review in their voice within hours.

1

Review Poller — n8n Workflow

Polls the Google Places API every 2 hours for new reviews. Checks against last_checked timestamp stored in Airtable. Triggers only when a new review appears — no unnecessary API calls.

2

Claude AI — Reply Generation

Claude reads the review (star rating, text, reviewer name). Generates a reply that sounds human — warm, specific to the review content, written in the pub's established voice. Never generic. Never corporate.

3

Google My Business API — Auto-Post

Reply is posted directly to the Google Business Profile via OAuth-authenticated API call. Post ID is logged. Owner's Google account is used — reply appears as coming from the pub, not from Zero Clicks.

4

Telegram Notification

Owner receives a Telegram message showing: the original review, the star rating, and the reply that was just posted. They can see everything but don't have to do anything. 1★–2★ reviews are flagged immediately for owner awareness.

n8n Workflow: "Zero Clicks — Reputation Manager"
──────────────────────────────────────────────
Cron Trigger (every 2hrs)
  → Google Places API: fetch reviews
  → IF new review since last_checked:
      → Claude API: generate reply (pub voice)
      → Google My Business API: post reply
      → Airtable: update last_checked
      → Telegram: notify owner
  → Loop to next client

Growth Product

GBP Footfall Engine

Inspired by Dusk (app that drove £2.7M incremental pub spend in summer 2023 via free drink offers). Same model — without splitting revenue with a third-party app.

"AI writes the message. The pub controls the economics."

AI is completely banned from inventing offers or discounts. The pub owner pre-approves a library of safe, food-led offers. AI only writes the caption — never the price, never the deal, never the alcohol mention.

3-Lane Content Strategy

Lane A — Promotion Posts: Pre-approved deals (Burger + Side, 10% off, etc.)

Lane B — Event Posts: Live music, quiz night, sports screening, Sunday roast

Lane C — Freshness Updates: Terrace open, new menu, kitchen open late

Rotating lanes keeps the algorithm active and compliance risks low.

3-Layer Verification

Technical: API returns 200, post published, post ID stored

Platform: Post visible on mobile Google Maps, survives moderation

Commercial: Coupon code tracked at POS — attributable revenue, not vanity metrics

Without all three layers, we can't defend the £95/month charge.
